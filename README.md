# CortexAI 🤖

A full-stack conversational AI chatbot built using React, FastAPI, OpenRouter, and Google Gemini 2.0 Flash.

## Features

* Multi-turn conversations
* Context-aware responses
* FastAPI backend
* React frontend
* Gemini 2.0 integration via OpenRouter
* Real-time chat interface

## Tech Stack

* React + Vite
* FastAPI
* Python
* OpenRouter
* Google Gemini 2.0 Flash

## Setup

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Architecture

React → FastAPI → OpenRouter → Gemini → FastAPI → React
